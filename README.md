# Sonai Kamalam Educational Trust Website

Official verification-ready website for **Sonai Kamalam Educational Trust** and its educational institution, **Senthamarai College of Arts and Science**.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Netlify Publish (Ready)

This project is pre-configured for Netlify with:

- `netlify.toml` (build command, publish directory, SPA redirects)
- `public/_redirects` for React Router fallback

### If deploying this folder directly (`senthamarai`)
No extra settings needed.

### If connecting the full repo in Netlify (monorepo setup)
Set these in Netlify site settings:

- **Base directory:** `senthamarai`
- **Build command:** `npm run build`
- **Publish directory:** `dist`
